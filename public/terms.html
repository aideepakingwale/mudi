02 Description of Service

MuDi ("the Service") is a web application that enables users to share and synchronise audio playback across multiple devices in real time. The Service provides:

  • Room creation and management for audio synchronisation sessions
  • Peer-to-peer file transfer of audio files between room participants
  • Real-time playback synchronisation using NTP clock alignment and WebRTC
  • User account creation and authentication
MuDi operates as a signalling and synchronisation service. Audio files are transferred directly between participants' devices using encrypted WebRTC DataChannels. The Service does not store, process, or retain audio file content.
